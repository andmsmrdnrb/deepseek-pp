## DeepSeek++ 1.0.9

1.0.9 把侧边栏聊天入口扩展到普通网页，并修复自动化任务在后台运行时的登录和会话恢复问题，重点让跨页面使用和定时任务更稳定。

### 主要变化

- **全局悬浮聊天入口**：非 DeepSeek 网页会显示可拖动的 DS++ Chat 悬浮入口，点击即可打开轻量聊天窗口，适合在浏览网页时直接向 DeepSeek++ 提问。
- **可控与隐私边界**：悬浮聊天可在设置页外观区域关闭；入口本身不会自动读取或发送页面正文，只有用户主动输入、发送或使用选中文本场景时内容才进入聊天流程。
- **自动化登录稳定性**：自动化任务在后台运行时会使用可用的 DeepSeek 登录信息，已登录用户不再因为后台环境拿不到页面 token 而误报需要重新登录。
- **自动化会话恢复**：自动化结果里的会话链接和历史快照在后台运行时会指向正确的 DeepSeek 会话，便于打开和追踪任务结果。
- **回归覆盖**：发布验证继续覆盖编译、测试、自动化契约、MCP/Shell/PoW smoke、多浏览器构建和发布资产检查。

### Chrome Web Store 更新说明

中文：

- 新增普通网页上的 DS++ Chat 悬浮入口，点击即可打开轻量聊天窗口；入口可拖动，也可在设置页外观区域关闭。
- 悬浮入口不会自动读取或发送页面正文；只有用户主动输入、发送或使用选中文本场景时内容才进入聊天流程。
- 修复自动化任务在后台运行时误报 DeepSeek 登录 token 缺失的问题，已登录用户可更稳定运行定时或手动自动化。
- 修复自动化结果里的会话链接和历史快照在后台环境下指向异常的问题。

English:

- Adds a DS++ Chat floating launcher on normal web pages, opening a lightweight chat window on click; the launcher is draggable and can be turned off from Appearance settings.
- The floating launcher does not automatically read or send page body content; content enters chat only when the user types, sends, or uses a selected-text scenario.
- Fixes background automation falsely reporting a missing DeepSeek login token, so signed-in users can run scheduled or manual automation more reliably.
- Fixes automation result session links and history snapshots resolving incorrectly from the background environment.
