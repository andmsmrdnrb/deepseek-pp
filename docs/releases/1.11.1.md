## DeepSeek++ 1.11.1

1.11.1 是一次聚焦 MCP 工具连接可靠性和可控性的更新：兼容更多仍在使用的服务版本，修复工具失败后影响后续对话的问题，并为大型工具集提供更清晰的管理方式。

### 主要变化

- **MCP 连接兼容**：兼容更多仍在使用的 MCP 服务版本，减少连接成功但工具不可用的情况。
- **工具调用恢复**：MCP 工具发现或调用遇到问题后，可通过刷新和后续对话恢复；工具结果返回后可继续推进同一轮任务。
- **大型工具集管理**：单个 MCP 服务可选择直接展示、按需适配或按需调用，更适合工具数量较多的服务，同时保留用户对实际工具的控制。
- **一致的执行体验**：手动对话、Agent、侧边栏和自动化任务使用一致的工具可用性与授权规则。
- **权限变化**：Chrome、Edge 和 Firefox 均不新增浏览器权限。

### 验证

- `npm run ci:quality`
- `npm pack --dry-run --workspace packages/shell-host`
- `node scripts/web-search-smoke.mjs`
